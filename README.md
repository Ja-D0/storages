# Storages

Переиспользуемые PHP-абстракции и реализации хранилищ.

## Совместимость

Пакет рассчитан на PHP `7.2.34+` и PHP `8.x`, включая PHP `8.2.19`.

## Что входит

- базовые интерфейсы для чтения и записи объектов;
- адаптеры для S3, OpenStack Swift/Object Store и локальной файловой системы;
- фабрики клиентов и хранилищ;
- минимальная реализация кластера поверх нескольких Storage-узлов.
