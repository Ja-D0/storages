# Storages

Переиспользуемые PHP-абстракции и реализации хранилищ.

## Совместимость

Пакет рассчитан на PHP `7.2.34+` и PHP `8.x`, включая PHP `8.2.19`.

## Что входит

- базовые интерфейсы для чтения и записи объектов;
- интерфейс и DTO для листинга объектов по префиксу;
- адаптеры для S3, OpenStack Swift/Object Store и локальной файловой системы;
- фабрики клиентов и хранилищ;
- минимальная реализация кластера поверх нескольких Storage-узлов.

## Листинг объектов

Все встроенные реализации `Storage` поддерживают получение списка объектов по префиксу:

```php
foreach ($storage->listObjects('documents/2026/') as $object) {
    echo $object->path;
    echo $object->basename;
    echo $object->size;
}
```

`ClusterStorage` обходит узлы в порядке их передачи в конструктор, пропускает недоступные узлы и дедуплицирует объекты по `path`. Если один и тот же путь найден в нескольких узлах, возвращается объект из первого приоритетного узла.
